If you’ve already started with Colt’s courses and they are working for you, I’d stick with that.

The order I currently recommend for people after looking at the introductory material for Colt’s Courses is:

  1. Complete Web Developer Bootcamp
  2. MySQL Bootcamp (and only use SQL for databases after this, Mongo isn’t used by most companies)
  3. React Bootcamp (this is an excellent course)
    1. Build a non-trivial full-stack project here. Something like a working eCommerce site, or a basic social media site. Be sure to use common patterns such as MVC to structure your code. Your stack should be Node/Express/SQL (either MySQL or PostgreSQL) and React with React Bootstrap or Material UI for styling

  4. JavaScript Data Structures and Algorithms

If your goal is to find work, you’ll be ready. The courses above will give you a strong skill set for a junior dev.
